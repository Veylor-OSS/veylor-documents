# Veylor Documents

## Getting started
### Download project
```bash
git clone https://github.com/veylor-hq/veylor-documents
cd veylor-documents
```

### Install dependencies
```bash
yarn
```

### Run Dev Mode
Web:
```bash
yarn dev
```
App:
```bash
yarn tauri dev
```
